# Student Academic Performance Dashboard

## 📌 Project Overview
An educational analytics dashboard monitoring student grades, attendance trends, and behavioral statistics to help academic advisors support struggling students.

## 🎯 Key Questions Answered
*   Is there a direct correlation between class attendance rates and final test scores?
*   Which subjects have the highest failure rates?
*   How do grades vary across different demographic groups?

## 🛠️ Data & Tech Stack
*   **Data Source**: School management system test scores and attendance logs.
*   **Transformations**: Created conditional formatting rules to highlight students-at-risk.
*   **DAX Formulas**: Built measures for Average Attendance Rate, Passing Rate %, and Grade Bins.
