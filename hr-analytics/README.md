# HR Attrition & Employee Demographics Dashboard

## 📌 Project Overview
This human resources dashboard provides HR managers with insights into employee retention, turnover trends, and team demographic distributions.

## 🎯 Key Questions Answered
*   What is the current employee attrition rate?
*   Which departments experience the highest turnover?
*   What is the distribution of employees by age, gender, and tenure?

## 🛠️ Data & Tech Stack
*   **Data Source**: HR information system database exports.
*   **Transformations**: Structured tenure groupings and age bins.
*   **DAX Formulas**: Formulated calculations for attrition rate percentage, active employee count, and average satisfaction score.
