# E-commerce Sales & Performance Dashboard

## 📌 Project Overview
An interactive e-commerce dashboard tracking sales metrics, profit margins, and shipping logistics. It provides retail managers with insights into geographical sales distributions and inventory status.

## 🎯 Key Questions Answered
*   Which product categories drive the highest profit margins?
*   What are the shipping delays/processing times per logistics partner?
*   What is the Month-over-Month (MoM) revenue growth?

## 🛠️ Data & Tech Stack
*   **Data Source**: E-commerce order details, customer profiles, and logistics tracking.
*   **Transformations**: Normalized date tables for time-intelligence calculations.
*   **DAX Formulas**: Built measures for YoY Sales Growth, Total Orders, Average Order Value (AOV), and Return Rates.
