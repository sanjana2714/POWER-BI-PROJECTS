# Banking Sales Analysis Dashboard

## 📌 Project Overview
This dashboard analyzes customer acquisition, account distributions, and sales agent performance for a commercial banking institution. It helps executives identify high-performing loan products and branch locations.

## 🎯 Key Questions Answered
*   Which loan products generate the highest revenue?
*   How does sales performance vary across branches and agents?
*   What is the trend of customer deposits over time?

## 🛠️ Data & Tech Stack
*   **Data Source**: Banking transactional and CRM data.
*   **Transformations**: Cleaned null values and merged customer tables with sales agent tables using Power Query.
*   **DAX Formulas**: Designed customized measures for customer acquisition cost (CAC), average loan values, and sales targets.
