# Power BI Analytics Portfolio

Welcome to my Power BI Portfolio! This repository contains a collection of interactive business intelligence dashboards and data visualization projects. These projects demonstrate my ability to ingest, transform (Power Query/M), model (DAX), and visualize complex datasets to drive data-driven decision making.

## 🛠️ Skills Demonstrated
*   **Data Preparation**: Data cleaning, merging, and reshaping using **Power Query** (M formula language).
*   **Data Modeling**: Building relational schemas, defining active/inactive relationships, and implementing star/snowflake schemas.
*   **Calculations & Modeling**: Designing custom columns and measures using **DAX (Data Analysis Expressions)** (e.g., CALCULATE, DIVIDE, RELATED, and Time Intelligence functions).
*   **Data Visualization**: Designing intuitive, user-friendly layouts, implementing tooltips, bookmarks, page navigation, and conditional formatting.

## 🚀 How to View
To interact with these dashboards:
1. Download and install **Power BI Desktop** (free).
2. Clone or download this repository.
3. Open any `.pbix` file inside its respective project folder.
