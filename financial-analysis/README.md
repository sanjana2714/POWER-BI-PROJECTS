# Financial Performance & Budget Analysis Dashboard

## 📌 Project Overview
A financial intelligence dashboard tracking company expenses, revenues, and budget allocations to support CFOs in monitoring overall fiscal health.

## 🎯 Key Questions Answered
*   What is the variance between actual expenses and the allocated budget?
*   What are the primary cost drivers within operating expenses (OPEX)?
*   How does the current net profit margin compare to the previous year?

## 🛠️ Data & Tech Stack
*   **Data Source**: General ledger exports and budget spreadsheets.
*   **Transformations**: Created structural parent-child hierarchies for financial accounts.
*   **DAX Formulas**: Built measures for Actuals YTD, Budget YTD, and Budget Variance %.
